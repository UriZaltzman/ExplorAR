# Carpeta de imágenes para la actividad Matea

Coloca aquí las imágenes que quieres agregar a la actividad "Matea experience".

## Instrucciones:

1. Coloca las imágenes en esta carpeta (`scripts/images/matea/`)
2. Formatos soportados: jpg, jpeg, png, gif, webp
3. Puedes nombrar los archivos como quieras (ej: foto1.jpg, imagen1.png, etc.)
4. Ejecuta el script: `node scripts/upload_and_add_photos_matea.js`

El script leerá automáticamente todas las imágenes de esta carpeta y las agregará a la actividad.
